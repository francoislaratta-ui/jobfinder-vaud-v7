# Installation

## Cloner le dépôt

```bash
git clone https://github.com/votre-compte/jobfinder-vaud-v13.git
```

## Entrer dans le dossier

```bash
cd jobfinder-vaud-v13
```

## Installer les dépendances

```bash
npm install
```

## Lancer le serveur

```bash
npm start
```
